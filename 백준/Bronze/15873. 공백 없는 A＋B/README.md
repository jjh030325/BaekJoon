### 문제 설명

<p>자연수 A, B가 주어지면 A+B를 구하는 프로그램을 작성하시오.</p>

### 입력 

 <p>자연수 A, B (0 < A, B ≤ 10)가 첫 번째 줄에 주어진다. 단, 두 수의 사이에는 공백이 주어지지 않는다. 두 수의 앞에 불필요한 0이 붙는 경우는 없다.</p>

### 출력 

 <p>첫 번째 줄에 A+B의 값을 출력한다.</p>
